# Software Personnel Management System

A C++ based employee management system developed using Object-Oriented Programming and File Handling.

## Features
- Add Employee
- View Employees
- Search Employee
- Update Employee
- Delete Employee
- Automatic File Saving

## Technologies Used
- C++
- OOP Concepts
- Text File Storage

## How to Run

Compile:

```bash
g++ -std=c++17 app.cpp -o app
```

Run:

```bash
./app
```

## Project Modules
1. Employee Management
2. File Handling
3. Search and Update
4. Delete Records
